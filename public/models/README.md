# AR Glasses Models

Папка для локальных **реальных** 3D-моделей очков (`.glb`) для AR примерки.

Используются такие имена файлов:

- `classic-aviator.glb`
- `modern-round.glb`
- `sport-pro.glb`
- `vintage-square.glb`
- `designer-cat-eye.glb`
- `urban-wayfarer.glb`

В `Products.tsx` эти локальные файлы стоят **первыми** в списке URL, поэтому если файл есть в `public/models`, он будет загружен раньше внешних CDN-ссылок.

Требования к модели для лучшей посадки:

- центр модели в районе переносицы,
- модель ориентирована вперёд по оси Z,
- реальный масштаб оправы (межзрачковое ~6-7 см условно в модели),
- умеренное число полигонов для real-time трекинга.
